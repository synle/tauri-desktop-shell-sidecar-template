# tauri-desktop-shell-sidecar-template

Skeleton Tauri v2 desktop app template with a shell binary sidecar. React 19 + MUI v9 + TypeScript on the frontend; Rust + `@tauri-apps/plugin-shell` for the sidecar process.

## Quick Start

```bash
npm ci || npm install --no-fund --prefer-offline
```

```bash
npm run tauri:dev
```
